# NanoPilot Shield

NanoPilot Shield is a custom Arduino Nano expansion PCB designed for robotics, embedded systems, and control applications. The board provides multiple servo interfaces, I2C communication ports, SPI connectivity, onboard status LED indication, and a compact shield-style layout for easy sensor and actuator integration.

Designed using KiCad, the shield enables rapid prototyping for motion control, autonomous systems, sensor interfacing, and embedded hardware development.

## Features
- 6 Servo output connectors
- 4 I2C communication ports
- SPI interface header
- Onboard power/status LED
- Compact Arduino Nano shield design
- Dedicated pull-up resistors for I2C communication
- Decoupling capacitor for stable operation
- Easy sensor and module integration

## Hardware Interfaces

### Servo Headers
Supports direct connection of servo motors for motion control applications.

### I2C Ports
Compatible with:
- MPU6050
- BMP280
- OLED Displays
- Other I2C Sensors

### SPI Header
Provides SPI communication support for high-speed peripheral devices.
